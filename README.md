# Proyecto Integrador: WordPress en Entorno Docker

Este repositorio contiene la configuración completa para desplegar un sitio web de WordPress utilizando **Docker** y **Docker Compose**, asegurando un entorno de desarrollo y pruebas modular y reproducible. El proyecto consiste en el servicio web (WordPress) y el servicio de base de datos (MySQL).

---

## Requisitos Previos

Para poder ejecutar este proyecto, necesitas tener instalados los siguientes programas en tu sistema:

1.  **Git:** Para clonar el repositorio.
2.  **Docker Desktop** (o Docker Engine): Para manejar los contenedores.
3.  **Docker Compose:** Para gestionar el entorno multi-contenedor (generalmente incluido con Docker Desktop).

---
